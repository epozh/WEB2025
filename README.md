# Интернет-магазин спортивных товаров

## Описание
Простой интернет-магазин с функционалом:
- Просмотр каталога товаров
- Добавление/удаление товаров в корзину
- Изменение количества товаров
- Сохранение корзины в localStorage
- Оформление заказа через модальное окно

## Технологии
- HTML5 (семантическая разметка)
- CSS3 (Flexbox, Grid, медиазапросы)
- JavaScript (DOM, localStorage, обработка форм)

## Структура
- `index.html` — основная страница
- `style.css` — стили
- `script.js` — логика
- `.gitignore` — игнорирование служебных файлов
- `README.md` — это файл
- `images/` — папка для изображений (по условию задачи)
- `favicon.ico` — иконка сайта

## Все требования соблюдены: 
- чистый HTML/CSS/JS
- семантика
- адаптивность
- localStorage
- модальные окна
